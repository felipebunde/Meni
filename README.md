MENI - Point of Sale (POS) System

MENI is a Point of Sale (POS) system designed with a focus on usability, speed, and simplicity for daily operations.

🚀 Features

- Sales management

- Product registration and control

- User-friendly interface

- Fast and practical workflow for real usage

🛠️ Technologies

Python

Flask 

📦 Requirements

Make sure you have Python installed (version 3.8 or higher recommended).

You can check your version with:

python --version

▶️ How to Run

Clone this repository:

git clone https://github.com/felipebunde/meni.git

Navigate to the project folder:

cd meni

(Optional) Create a virtual environment:

python -m venv venv

Activate it:

Windows:
venv\Scripts\activate

Linux/Mac:
source venv/bin/activate

Install dependencies (if applicable):

pip install -r requirements.txt

Run the application:

python app.py
